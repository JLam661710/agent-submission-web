**核心目的**
- 从“自身与附近”出发，挖掘可落地的 Coze 智能体/工作流选题与学习路径。
- 为社群互动与信息收集服务，帮助填写者形成“选题—MVP—学习—挑战—评价”的闭环认知。
- 降低无目的学习与信息噪音，把零散想法转为可验证的轻量方案。

**主要功能**
- 提供包含13题的多步问卷，支持进度指示与本地草稿保存。
- 提交后，后端写入飞书多维表（Bitable），并调用 Coze API 生成分析文本。
- 结果页仅展示 Coze 分析报告（不回显原始答案），如“相关性话题报告”或“选题—MVP—路径—挑战—评价”简报。
- 防重复提交、统一错误提示，移动端友好。

**为什么值得做**
- 将个人技能、烦恼与身边/社区挑战建立映射，快速锁定可执行选题。
- 提供最小数据流与MVP草图，便于实践与复盘；数据沉淀到 Bitable 便于后续分析。
- 以轻量流程提升方向确定效率与信息一致性，减少试错成本。

**适用人群与场景**
- 正在探索 Coze 智能体/工作流的创作者、学生、职场人士、社区参与者。
- 用于社群活动/工作坊、团队内方向梳理、个人学习路径规划。

**边界与非目标**
- 不要求登录；不采集行为数据；不做后台问卷配置（MVP阶段）。
- 不提供商业化建议或复杂技术实现指导，仅输出结构化分析与路径提示。

**成功度量**
- 时间节省（选题与方向确定更快）。
- 信息一致性（个人—团队—社区口径更统一）。
- 复用率（方案与数据在 Bitable 的可复盘与再利用率）。
